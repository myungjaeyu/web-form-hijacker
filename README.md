## Install

```bash
$ git clone https://github.com/u4bi-git/web-form-hijacker.git hi
```

```bash
$ cd hi
```

```bash
$ npm install
```

## Usage

```bash
$ npm run server
```

*[Download ngrok (Because of SSL connection)](https://ngrok.com/download)*
```bash
$ ./ngrok http 7777
```

*Load unpacked extension (chrome://extensions)*
